Política de Privacidade

Unified Calendar Sync

Unified Calendar Sync não recolhe, não armazena nem transmite quaisquer dados pessoais.

Todos os dados de calendário permanecem no dispositivo do utilizador e são processados localmente com o framework EventKit da Apple. Nenhum dado é enviado para servidores externos e não são utilizadas tecnologias de análise ou de rastreio.

A app apenas acede aos dados de calendário que o utilizador autoriza explicitamente através das permissões do sistema macOS.
