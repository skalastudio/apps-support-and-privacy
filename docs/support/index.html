Skala Studio Support

Email support for Skala Studio apps.

Contact

If you need help, have questions, or want to report an issue, email us at:

skalastudio.apps@gmail.com

What to include

  • App name and version
  • macOS version
  • Account providers involved (iCloud, Exchange, Google, etc.)
  • Steps to reproduce the issue
  • Relevant screenshots or error messages

Response time

We reply as quickly as possible, typically within a few business days.
