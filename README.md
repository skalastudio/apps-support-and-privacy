# Skala Studio Support & Privacy

This repository hosts the public support and privacy policy documents for Skala Studio apps. It exists as a temporary home until a dedicated website is available and is linked from App Store Connect.

## Structure
- `docs/support/` contains support pages by locale, named `support-<locale>.html` (for example, `docs/support/support-en-US.html`).
- `docs/apps/<app>/privacy/` contains privacy policies by product and locale (for example, `docs/apps/unified-calendar-sync/privacy/privacy-en-US.html`).
- `docs/apps/<app>/marketing/` contains the marketing page for each app (for example, `docs/apps/unified-calendar-sync/marketing/index.html`).
